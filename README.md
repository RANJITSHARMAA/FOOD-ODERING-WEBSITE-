 .... Foodie – Online Food Ordering Website
..... Project Overview

Foodie is a responsive online food ordering website built using HTML5, CSS3, and JavaScript. It provides a simple and user-friendly interface for browsing food items, searching dishes, adding products to a shopping cart, and placing orders.

..... Features
Responsive design for desktop, tablet, and mobile devices
Attractive homepage with hero section
Food categories and popular menu items
Search functionality
Add to Cart using Local Storage
Cart item counter
Special offers section
Contact form
Smooth scrolling navigation
Basic login and signup pages
Modern UI with animations
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Font Awesome
Google Fonts
